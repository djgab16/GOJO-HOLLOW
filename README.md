# GOJO-HOLLOW
# Hand Gesture Hollow Effects

A sleek and professional real-time hand gesture detection program built with Python, OpenCV, and MediaPipe. 
This project brings Jujutsu Kaisen's iconic "Hollow" techniques to life through interactive webcam tracking!

## Features
- **Blue Hollow Effect:** Point your right index finger upward.
- **Red Hollow Effect:** Point your left index finger upward.
- **Purple Hollow Effect:** Pinch your thumb and index finger together (Thumb-Index Hold) with any hand.

## Requirements
Ensure you have Python installed (preferably Python 3.8+). Then install the required dependencies:

```bash
pip install opencv-python mediapipe numpy
```

## How to Run
Navigate into this folder and start the Python script:

```bash
python hand_gesture_hollow.py
```

## Controls and Usage
- Ensure your webcam is connected and permissions are granted.
- The system automatically mirrors the image to make it intuitive.
- Hold up your hands in front of the camera and experiment with the gestures!
- Make sure that other fingers are curled inward for the gestures to register correctly without flickering.
- Press `q` or `ESC` while focusing on the video window to safely quit the application.
